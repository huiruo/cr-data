# forsigner's tool

### Install

``` bash
$ npm i -g forsigner
```

### Alias

In your ```~/.bashrc ``` or ```~/.zshrc ``` add ```alias f='forsigner'```

### Dotfiles

Download dotfiles to current dir.

```bash
$ f get .eslintrc
```

You can get those dotfiles:

```bash
├── .babelrc
├── .bowerrc
├── .csscomb.json
├── .editorconfig
├── .esformatter
├── .eslintignore
├── .eslintrc
├── .gitconfig
├── .gitignore
├── .htmlhintrc
├── .jscsrc
├── .tern-project
└── .zshrc
```

$ f ip
